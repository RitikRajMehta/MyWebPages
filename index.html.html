SOFTWARE

Software is a part of a computer,which cannot be touched.To make the hardware work,we need to have software installed in it.

Types of software
  1. System Software

    2. It is a set of program that controls overall working of a computer. For Example Operating System

  2. Application Software

    3. The software that can help you to specific type of work is called application software.For Example Microsoft word,Power Point,Ms Point.
